Como encomendar as mudas

Realizamos envios de mudas para todo o Brasil, geralmente usando os serviços da Azul Cargo. Para avaliar o custo do frete, é preciso ter uma estimativa das mudas a enviar e conhecer qual o município de destino.

Favor enviar um email para mudas.tamaras@gmail.com informando as mudas desejadas assim como o municípo (e/ou CEP) para estimativa do valor do frete

As mudas são enviadas com uma quantidade mínima de terra e umidade suficiente para minimizar o estrese das raizes.
Ainda que as tamareiras suportam muito bem o transplante e que são muito resistentes, se recomenda ter o local de plantio já preparado para poder localizar as mudas recebidas o antes posível.
